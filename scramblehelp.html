Scramble Coding Help

  • Steps:

    1. ARRAYS

      2. Create new arrays that contain the scramble notation.
      Use, "$:", to define a new array.
      Separate each scramble notation with ",".
      EX:
      • $:F,B,R,L,U,D

    2. VARIABLES

      3. Create new variables that are based on the arrays.
      Use, "@c", to define a new array, and "c" being any alphabetical character
      • It can be any symbol but using wildcards can confuse the program
      In "{}" brackets, define which array to pull notation from. (First array starts at 0)
      In "[]" brackets, define which specific notations are going to be pulled. (First notation starts at 0)
      • Separate each new entry with a ","
      • Use, "e", to pull every notation from an array
      In "()" brackets, define its scramble type
      l list every notation item
      n# Randomly choose a notation item, repeated for #, but same notation won't repeat after each other
      o Randomly list every notation item, but same notation won't repeat after each other
      r#1-#1 Randomly choose a notation item, repeated for any number between #1(min) and #2(max), items won't be repeated
      s#1-#2 Randomly choose a notation item, repeated for any number between #1(min) and #2(max), but same notation won't repeat after each other
      • Same Notation: first letter of a notation item won't repeat
      EX:
      • @a{0}[0,1,3](u10)
        • Variable "a", now pulls from array 0 (Array 0 was set to F,B,R,L,U,D), takes items number 0, 1, and 3 (F,B, and L), and randomly chooses one item 10 times

    3. PATTERNS

      4. Use, "%:", to create a new pattern.
      Call variables in any order.
      EX:
      • If we had created a variable called, "a", "b", and "c" and we made the pattern "abca", it would call the variable a, then b, then c, then a again.

  • EXAMBLE CODE FOR PYRAMINX

    • INPUT
    $:B,R,L,U,B',R',L',U'
    $:b,r,l,u,b',r',l',u'

    @a{0}[e](n11)
    @t{1}[e](s0-4)

    %at
    OUTPUT
    *not seen*
    Variable a = B,R,L,U,B',R',L',U'. randomize 11 times
    Variable t = b,r,l,u,b',r',l',u'. random number between 0 and 4

    a = B' L B U' R L' B' U L R' U

    random number is set to 2
    t = l' u'

    a + t

    *what is seen*
    B' L B U' R L' B' U L R' U l' u'

  • NOTES

    • Each piece of code needs its own line
    Space are not recognized, so items ca be serated with spaces
    Multiple patterns can be created, and are activated in order
    Text is ignored, so you can freely write comments and text on new lines
